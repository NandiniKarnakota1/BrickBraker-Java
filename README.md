# 🎮 Brick Breaker Game - Java Swing

A desktop-based Brick Breaker game developed using Core Java and Java Swing.  
This project demonstrates strong understanding of OOPS principles, event handling, real-time game logic, and GUI development.

---

## 🚀 Features

- Interactive paddle movement using keyboard controls
- Real-time ball movement with collision detection
- Brick breaking mechanics with score tracking
- Game Over and Winning conditions
- Restart functionality using ENTER key
- Smooth gameplay using Java Timer

---

## 🕹️ Controls

- ⬅ Left Arrow  → Move paddle left  
- ➡ Right Arrow → Move paddle right  
- ⏎ Enter       → Restart game  

---

## 🏗️ Technical Implementation

- Built using Java Swing for GUI rendering
- Used JPanel for custom painting
- Implemented KeyListener for user input handling
- Used Timer for game loop execution
- Implemented collision detection using Rectangle class
- Applied OOPS principles for structured and modular code
- Managed dynamic brick layout using MapGenerator class

---

## 🛠️ Tech Stack

- Java
- Java Swing
- AWT Graphics
- OOPS Concepts
- Event Handling
- Timer-based Game Loop

---

## 📂 Game Logic Highlights

- Ball direction changes based on paddle and brick collision
- Score increases dynamically when bricks are destroyed
- Game ends when:
  - All bricks are destroyed (YOU WON)
  - Ball crosses paddle boundary (GAME OVER)
- Game state resets on restart
